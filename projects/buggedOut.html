Bugged Out: Bracky's Game Jam 2023

Overview

Remember that semicolon you missed? Or the bracket you forgot? All the bugs in the code have warped time and space, sucking our developer, "the Dude," into the world of his own code. Now, faced with monsters and enemies generated by these bugs, he must fight to escape this buggy dimension.

Bugged Out is a 2D platformer action game where you fulfill your innermost desires by crushing bugs in your code and ensuring a clean, error-free document!

Created for Brackey's Game Jam 2023, the game placed 3rd out of 907 entries for innovation.
